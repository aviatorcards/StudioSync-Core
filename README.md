# StudioSync Core

This is a standalone package containing the core ERP modules for the StudioSync suite.

## Included Apps

- **core**: User management, studios, and shared models.
- **billing**: Invoicing and payment processing.
- **lessons**: Scheduling and lesson management.
- **students**: Student and family tracking.
- **resources**: Knowledge base and file management.
- **messaging**: Internal messaging system.
- **notifications**: Multi-channel notification engine.
- **inventory**: Equipment and asset tracking.
- **auth**: Custom JWT-based authentication.
